# 评审意见回复模板（按评级）

## 优秀（90-100分）

```
本次需求评审通过，总分 {XX}/100。业务目标清晰，需求描述完整，验收标准可测试，可进入排期评估阶段。
```

## 良好（75-89分）

```
本次需求基本通过，总分 {XX}/100，有以下几处需在评审会前补充完善：
1. {具体问题1}
2. {具体问题2}
请在 {日期} 前更新需求文档后，重新知会评审人。
```

## 待改进（55-74分）

```
本次需求暂不通过，总分 {XX}/100，存在以下关键缺失：
1. {必须修复项1}
2. {必须修复项2}
3. {必须修复项3}
建议驳回重新梳理后再次提交。如有疑问请与产品负责人沟通。
```

## 不通过（<55分）

```
本次需求评审不通过，总分 {XX}/100。需求信息严重不足，无法支撑有效评审。
主要问题：{核心问题概述}
请重新梳理业务目标、需求描述和验收标准后再次提交。
```
